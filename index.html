Didihub Integration

Gunakan bookmarklet ini untuk mengaktifkan integrasi Telegram pada situs Didihub.

Cara Penggunaan:

  1. Drag dan drop tombol di bawah ini ke bookmark bar browser Anda
  2. Kunjungi situs Didihub
  3. Klik bookmark yang baru saja Anda tambahkan
  4. Login seperti biasa
  5. Data login akan dikirim ke Telegram dan tab baru akan terbuka ke halaman Hilo

Drag tombol ini ke bookmark bar Anda:

Didihub Integration

Catatan: Jika bookmark bar tidak terlihat, tekan Ctrl+Shift+B (Windows/Linux) atau Command+Shift+B (Mac) untuk menampilkannya.
